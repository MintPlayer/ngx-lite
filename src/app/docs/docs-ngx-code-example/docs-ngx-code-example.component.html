ngx-code-example

The ngx-code-example component uses prismjs and Angular to easily display code examples in a view.

Demo

Installation

To install this library, run npm install @ngx-libraries/code-example --save and then from your Angular AppModule:

Component

Once the library is imported, you can use the ngx-code-example component. 

  
<ngx-code-example>
  <![CDATA[
if (true) {{'{'}}
  console.log('hi');
{{'}'}} else {{'{'}}
  console.log('bye');
{{'}'}}
  ]]>
</ngx-code-example>
